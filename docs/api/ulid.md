# llm_toolkit_schema.ulid

This module provides ULID (Universally Unique Lexicographically Sortable Identifier)
generation, validation, and timestamp extraction.

> **Auto-documented module:** `llm_toolkit_schema.ulid`

## Key functions

- `generate()` — generate a new ULID string
- `validate()` — check whether a string is a valid ULID
- `extract_timestamp_ms()` — extract the millisecond timestamp embedded in a ULID
